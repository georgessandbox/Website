Hobo Frogger

  • Powered by Processing graphical library. (processing.org)
  • The focus is to get adapted to an unfamiliar library and code a simple frogger-like game in a timely fashion.
  • The player's goal is to hop on different tracks to avoid incoming trains for the longest time possible (a timer is showed at the top)
  • The trains show up based on Poisson discrete probability distribution.
  • Hobos inside the tunnel might lie when shouting about coming train so be careful.

The game changes difficulty dynamically by removing tracks when a milestone is reached

As well as lowering game difficulty by adding more tracks when player is hit by too many trains too soon.
